## 스프링부트 로컬 로그인 템플릿

스프링 부트를 활용한 프로젝트시, 쉽게 적용할 수 있는 로그인 템플릿입니다.
로컬 부분만 해당되며, Oauth는 다른 브랜치에서 사용해야 합니다.


### 버젼

| Category  | Stack                |
|-----------|----------------------|
| Framework | - Spring Boot 2.7.12 |
| Test      | - JUnit 5            |
| security  | - spring security    |
| Database  | - MySql 8.0          |
