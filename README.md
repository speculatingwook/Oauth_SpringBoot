## 스프링부트 소셜 로그인(Oauth2.0) & 로그인 템플릿

현재 로컬 로그인까지 테스트가 완료된 프로젝트입니다. 프로젝트에 원활히 사용할 수 있도록 수정 작업을 완료하였고, 템플릿으로 가져다 쓰면 됩니다.
